# NYT_LetterBoxSolver
A simple search written in Julia to search two letter combinations to solve the NYTs Letter Box puzzle

## Letter Box Puzzle
The letter box is a daily puzzle in NYTs games [here](https://www.nytimes.com/puzzles/letter-boxed)
